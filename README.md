📚 EvolBooks

EvolBooks is a PHP MVC-based online book platform that allows users to discover, explore, and interact with books through a modern and responsive interface.

🚀 Features

- User Registration & Login
- User Profile Management
- Browse Book Library
- Book Details Page
- Like / Unlike Books
- Personalized Recommendation System
- Contact Us System
- Admin Dashboard
- Book Management (CRUD)
- User Management
- Contact Message Management
- Responsive Design for All Devices

🛠️ Technologies Used

- PHP
- MySQL
- Bootstrap 5
- JavaScript
- HTML5 & CSS3
- MVC Architecture
- PHPMailer

⚙️ Installation

1. Clone the repository:

git clone https://github.com/Diabloo10/evolbooks.git

2. Import the database:

database/evol.sql

3. Configure database credentials in:

app/core/Database.php

4. Place the project inside:

xampp/htdocs/

5. Start Apache and MySQL from XAMPP.

6. Open in browser:

http://localhost/evol/public

📂 Project Structure

app/        -> Controllers, Models, Views
public/     -> Assets and Entry Point
database/   -> SQL Database File
storage/    -> Uploaded Files
vendor/     -> PHPMailer Dependencies
